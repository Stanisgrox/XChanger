Настройки

Основная валюта

Вы можете задать основную валюту. Курсы валют на главной странице будут сверяться с выбранной вами основной валютой. Введите трехзначный код валюты (например: "RUB" - российский рубль).

Неправильный код валюты

API ключ

Для обновления базы данных приложения необходимо подключение к базе данных Open Exchange Rates (https://openexchangerates.org/). Здесь вы можете изменить ключ API.

Неправильный ключ API
Ключ API изменен. Теперь вы можете обновить базу данных.

Обновить базу данных

Дата и время последнего обновления базы данных показана на главной странице. Здесь вы можете загрузить актуальные данные с OpenExchangeRates. Для обновления данных необходимо заменить весь текст в файле data/exchange.json в папке приложения на текст, который был получен в окне, открывшемся после активации кнопки.

Отключить кеширование

При отключении кеширования, данные будут браться напрямую с OpenExchangeRates.
ВНИМАНИЕ: Без кеширования нет необходимости обновлять базу данных, но количество запросов в месяц ограничено. Кеш будет отключен только на эту сессию. Чтобы включить его снова, перезагрузите страницу.
